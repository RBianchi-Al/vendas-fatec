
## Replicar o projeto

1. Para iniciar o projeto usando Node.js
```sh
   npm init -y
    # ou
   yarn init -y

```
2. Instalando os pacotes 
```sh
    npm install
```  
3. Rodar migrations
```sh
    npm run migrate
```  
4. Rodar seeders
```sh
    npm run seed
```  
5. Executar a API localmente
```sh
    npm run dev
```
